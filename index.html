Sobre o Projeto

Este projeto foi criado para demonstrar a gestão de um time fictício no GitHub, com organização via Issues, Pull Requests e Project Board.

O objetivo deste projeto é demonstrar, na prática, como funciona a colaboração em equipe com GitHub: criação de Issues, organização em um Project (Kanban), uso de branches, Pull Requests e revisões.
